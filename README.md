# Otoco.io DApp Web Client

## Website URL
> https://otoco.io

## Hosting
> Netlify

## Development
> Dependency installation
```
$> yarn
```

## Build
```
$> yarn build:prod
```

### In Case of lack of memory
```
$> export NODE_OPTIONS=--max_old_space_size=4096
```

## Running project locally
```
$> yarn start:dev
```
Surf URL using browser with `Metamask`: http://localhost:8080
